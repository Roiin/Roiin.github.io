- **Editor:** Logic Node Editor
    
- **Node Group:** Math
    
- **Core Function:** Maps an input value to an output value based on a user-editable curve, allowing for non-linear transitions (easing).
    
- **Key Properties & Settings:**
    
    - **Input (Value):** The input factor, typically from 0.0 to 1.0.
        
    - **Setting (Curve Editor):** A graphical interface to shape the response curve.
        
    - **Output (Value):** The remapped output value.
        
- **Practical Application:** Smoothing animations or movements. Used for "ease-in/ease-out" effects, making a camera movement start slowly, accelerate, and then slow down at the end.
