- **Editor:** Logic Node Editor
    
- **Node Group:** Values
    
- **Core Function:** Outputs a random floating-point number within a specified range.
    
- **Key Properties & Settings:**
    
    - **Min (Socket):** Float input for the minimum possible value.
        
    - **Max (Socket):** Float input for the maximum possible value.
        
    - **Result (Socket):** The random float output.
        
- **Practical Application:** Used to introduce variation. Examples: randomizing the spread of bullets, adding slight variations to an enemy's movement speed, or determining the chance of a critical hit.
